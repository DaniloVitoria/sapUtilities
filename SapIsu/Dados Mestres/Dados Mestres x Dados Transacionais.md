
# Dados mestres:

Estão relacionados aos cadastros que serão utilizados pelo sistema:

Cadastro cliente, Endereços e etc

São dados que se mantém inalterados por um longo período de tempo e neste período é a única versão válida




# Dados Transacionais

São dados criados a execução dos processos comerciais (leitura, faturamento, instalção de equipamento, recebíveis)

Esses dados são dinâmicos sendo válidos durante um curto período de tempo. (cada mês tem uma diferente leitura e um diferente faturamento).
