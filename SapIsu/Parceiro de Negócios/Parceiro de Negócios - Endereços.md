
Endereços são armazenados no Parceiro de Negócio

Um mesmo parceiro de negócio pode termais de um endereço

Um dos endereços cadastrados deve ser declarado como **standard**
- Este endereço de correspondência será utilizado como modo default caso não haja um endereço específico indicado

Um endereço pode estar ligado a um número de telefone específico

O Parceiro de negócio utiliza o cadastro central de endereços para manutenção dos mesmos

Endereços para diferentes funções podem ser associados no nível conta contrato
